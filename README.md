> A personal challenge tracker for **Striver’s SDE Sheet** — the popular step-by-step DSA problem list for software engineering interviews.

## Overview

Striver’s SDE Sheet curates the most interview-relevant Data Structures & Algorithms problems into a structured path. This repository tracks my progress working through every problem in the sheet, organized by topic, as I prepare for SDE interviews.

## Sheet Topics

- 🔵 Arrays / Partitions
- 🟢 Linked Lists
- 🟡 Greedy / Recursion / Backtracking
- 🔴 Trees / Graphs
- 🟣 Dynamic Programming
- ⚫ Strings / Math / Misc

## How I Track Progress

Each problem is checked off as it's solved, with notes/approaches added over time. The goal is consistency — finishing the full sheet end to end.
